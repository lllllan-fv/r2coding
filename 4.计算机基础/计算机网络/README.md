## URL 和 URI



### 定义

- URL

> 统一资源**定位器**，它是WWW的统一资源定位标志，就是指网络地址。

- URI

> 统一资源**标识符**，统一资源标识符



### 联系 区别

- URL 是一种 URI
- URI就是一种资源定位机制。它是比較笼统地定位了资源，并不局限于client和server，
- 而URL就定位了网上的一切资源，仅仅要是网上的资源，都有唯一的URL.
